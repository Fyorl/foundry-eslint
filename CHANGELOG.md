## 1.0.2

### Improvements
 - Added a rule for no spaces around default params in JSDoc.
 - Added a rule for no spaces around default params in JS.
 - Removed no-spaces-in-union-types rule in favour of jsdoc/type-formatting option.
 - Added a rule for requiring braces around multi-line arrow-funcs.
 - Improved separator rule to handle loose functions.
 - Allow single-line `@inheritdoc` or `@override`.
 - Forbid hyphens between `@param` and `@property` names and descriptions.
 - Require brackets around all binary operators.
 - Added tests.
 - Added README.

### Fixes
 - Fixed require-arrow-braces auto-fixer formatting.
